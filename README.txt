== Description ==

= Business pack =

The Business Pro Pack includes a number of cool new drag and drop modules:

* Number counter
* Accordian
* Tabs
* Testimonials
* Google maps

== Installation ==

Installing "Business pack” can be done either by searching for "Business pack" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress
